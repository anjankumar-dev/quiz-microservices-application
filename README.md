🚀 Quiz-Microservices Application (Spring Boot)

📌 About the Project The Quiz-Microservices Application is a Spring Boot application built for creating a quiz for students and practiced How a microservices application built.
It provides REST APIs for CRUD operations, Basic microservices Architecture. Learing out comes from this project: building individual microservices and connecting them with a eureka service and integrated API Gatway for centralized routing.
Used DTOs for efficient data transfer.

✨ Features

Quiz-Service
1. creating a quiz
2. Getting the quiz by quizId
3. submitting the quiz

Question-Service
1. Retrieving all the questions
2. Retrieve all the quesions by category
3. Add a question
4. Generate questions for quiz
5. Calculating the score

Eureka Service Registry
API GateWay

🛠 Tech Stack

Backend: Java 17, Spring Boot 3.x
Database: MySQL
Build Tool: Maven
Service-Registry: Eureka Service, Open Fiegn
Cetralized Routing: API GateWay
